1️⃣ What is the difference between var, let, and const?
  ans: Variables can be created repeatedly under the name "var" and their values ​​can also be changed . If it is declared inside an if block, it can be accessed outside the block.

 "let" variable cannot be created twice with the same name in the same scope, but its value can be updated or changed. If it is declared inside an if block, it can be no accessed outside the block.

"const" variable This is used for variables whose value will never change. If it is declared inside an if block, it can be no accessed outside the block. Once a value is set, it cannot be changed and new variables cannot be created with the same name.
